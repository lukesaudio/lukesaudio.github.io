Luke does code?

You're darn right he do. While my background is in music performance, coming to unversity allowed me the chance to code in an audio environment, and we seem to have snowballed from there! I wouldn't say my programming sits in one particular area, but loosely in it'creative and often experimental application to audio, visual and interactive technologies.

I'm versed in C++, C# and HTML/CSS/JS, alongside frameworks such as openFrameworks, p5.js, JUCE and Unity scripting, and software such as Unity and Max/MSP. Below are a select few of my code pieces alongside some of the outputs created by the code.

Games

  • HackTheMidlands 4.0 - Reigning Cats and Dogs

    "Cats and Dogs fight to survive! A universe simulation in which cats roam free and dogs hunt them for food. Each animal battles for limited food resources across a beautiful terrain." An evolutionary simulator that sets cats and dogs to outlive one another on an island. This one is less polished as I had to leave early due to flooding and trains!

  • HackTheMidlands 2016 - The Web Wars

    "Send websites to their wwwdeath" - a SPONSOR-PRIZE-WINNING 2D auto-battler scraping website data using the Majestic API and using it to generate stats for two favicon-headed fighters!
